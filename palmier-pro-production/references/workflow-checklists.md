# Workflow Checklists

## Full Package Checklist

Use this sequence for a complete long-form plus Shorts project:

1. Intake: idea, audience, format, runtime, available assets, deadline, constraints.
2. Brief: viewer promise, hook, angle, title candidates, thumbnail tension, success metric.
3. Research: sources, proof points, viewer questions, risk notes.
4. Script: cold open, beats, narration, visuals, captions, music/SFX, citations.
5. Asset plan: user assets, generated assets, needed captures, filename plan.
6. Project folder: manifest, asset ledger, brief, script, edit plan.
7. Palmier import: bins, media import, A-roll/VO placement, b-roll placement, captions.
8. Edit passes: story, pacing, visual, audio, retention, compliance, export.
9. Shorts: candidate moments, 9:16 reframing, captions, filenames, export.
10. Packaging: title, thumbnail, description, chapters, tags, pinned comment.
11. Review: watchdown notes, fixes, final export, upload checklist.

## Intake Defaults

When the user gives only a rough idea, assume:

- one 8-12 minute YouTube video
- three Shorts cutdown ideas
- one thumbnail concept set
- one project folder under `./palmier-projects`
- Codex may generate scripts, image prompts, thumbnail concepts, edit plan, and metadata
- Codex should not publish anything without explicit approval

## Brief Template

Use this structure in `brief/creative-brief.md`:

- Working title
- One-sentence viewer promise
- Target viewer
- Why this is worth watching now
- Core curiosity gap
- Desired emotional payoff
- Format and runtime
- Required assets
- Generation plan
- Risks and approvals
- Success metric

## Script Quality Bar

- The first spoken line should matter.
- The first 30 seconds should show stakes, result, conflict, or contradiction.
- Each section should advance the central promise.
- The script should create visual opportunities every 5-15 seconds.
- Claims should be sourced, framed as opinion, or removed.
- The close should ask for one clear action.

## Thumbnail Quality Bar

- One dominant subject.
- One tension point.
- Reads on mobile in under two seconds.
- 0-4 words of text unless the user has a known channel style.
- Does not merely repeat the title.

## Shorts Quality Bar

- Hook in 1-2 seconds.
- One idea only.
- Captions are large and readable on mobile.
- No long setup.
- Clear payoff, loop, reveal, or comment prompt.

## Review Notes Template

Use this structure in `review/watchdown-notes.md`:

- File reviewed
- Watchdown date
- Audio issues
- Pacing issues
- Visual/caption issues
- Claims/rights/privacy risks
- Export issues
- Required fixes
- Optional improvements
