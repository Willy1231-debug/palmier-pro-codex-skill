# Palmier Pro Codex Skill

A production-ready Codex skill for creating long-form YouTube videos, Shorts, Reels, TikToks, thumbnails, scripts, and complete content workflows using Palmier Pro.

Instead of manually prompting an AI every time you want to edit a video, this skill gives Codex a structured production workflow that behaves like an experienced video editor and production assistant.

---

## Features

- 🎬 Long-form YouTube production
- ✍️ Script writing
- 🖼️ Thumbnail planning
- 📁 Automatic project organization
- 🎞️ Palmier Pro timeline workflow
- 📱 Shorts / Reels / TikTok generation
- 📦 Export planning
- 📝 Upload package generation
- 🔄 Repeatable production workflow
- 🧠 Uses Codex + Palmier MCP

---

## What It Does

The skill guides Codex through an entire production pipeline:

text Idea     ↓ Creative Brief     ↓ Research     ↓ Script     ↓ Asset Planning     ↓ Palmier Import     ↓ Timeline Editing     ↓ Review     ↓ Exports     ↓ Shorts     ↓ Thumbnail     ↓ Publishing Package 

Rather than writing dozens of prompts manually, Codex follows a consistent production workflow every time.

---

## Requirements

- Codex
- Palmier Pro
- Palmier MCP Server
- Palmier MCP configured in Codex

If Palmier tools are unavailable, the workflow automatically falls back to generating:

- scripts
- asset lists
- edit plans
- project folders
- timeline notes
- publishing packages

so work can continue.

## Installation

Clone this repository.

bash git clone https://github.com/<Willy1231-debug>/palmier-pro-codex-skill.git 

Copy the skill into your Codex skills directory.

Restart Codex.

The skill will automatically become available.

## Example

Example prompt:

> Create a 12-minute YouTube video announcing our latest product launch.

The skill will:

- generate the creative brief
- write the script
- organize assets
- prepare Palmier project structure
- create edit notes
- generate Shorts
- create thumbnail ideas
- prepare publishing assets
- FYI You will need to work with it, of course, to get your desired outcome a little bit. It’s not able to one-shot.
